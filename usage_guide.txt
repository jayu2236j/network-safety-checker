---------------------------------------------------------------------
1. Overview
---------------------------------------------------------------------
The Network Safety Checker is a Linux-based security audit tool. It performs a series of basic and advanced checks to help identify weak security configurations on a system. It also includes ARP-based Man-in-the-Middle detection, commonly found on public Wi-Fi networks. The tool provides a score out of 10 based on detected issues.

A log file is created after each run:
    /var/log/network_safety_checker.log

---------------------------------------------------------------------
2. What the Tool Checks
---------------------------------------------------------------------
The tool runs the following checks:

1. Firewall status (ufw or firewalld)
2. Open network ports
3. SSH security configuration
4. Accounts with empty passwords
5. World-writable files in /etc
6. Package update availability (APT)
7. Fail2Ban status
8. Dangerous legacy services (telnet, ftp, rlogin, rsh)
9. Home directory permissions
10. Kernel version check
11. Patch summary (APT simulation)
12. ARP spoofing / MITM detection

---------------------------------------------------------------------
3. Requirements
---------------------------------------------------------------------
To run this tool, you need:
- A Linux system
- Sudo access
- Git installed
- Internet is optional

---------------------------------------------------------------------
4. Downloading the Project
---------------------------------------------------------------------
Open a terminal and enter:

git clone <your_repo_url>
cd NetworkSafetyChecker

---------------------------------------------------------------------
5. Installation
---------------------------------------------------------------------
Make the installer executable:

chmod +x install.sh

Install the tool:

sudo ./install.sh

This creates a system-wide command called: netscan

---------------------------------------------------------------------
6. Running the Tool
---------------------------------------------------------------------
Run using the installed command:

sudo netscan

Run using the script directly:

sudo ./network_safety_checker.sh

View logs:

cat /var/log/network_safety_checker.log

---------------------------------------------------------------------
7. Using the Tool from a Live USB
---------------------------------------------------------------------
This tool is designed to work well from a Live USB session.

Steps:
1. Boot the computer using your Linux Live USB.
2. Open Terminal.
3. Download the project:

   git clone <your_repo_url>

4. Install it:

   chmod +x install.sh
   sudo ./install.sh

5. Run the scan:

   sudo netscan

This allows the USB to act as a portable security toolkit.

---------------------------------------------------------------------
8. Using the Tool on Windows through WSL
---------------------------------------------------------------------
Windows cannot run Linux scripts directly.  
To use this tool, install Windows Subsystem for Linux (WSL).

Install WSL:

wsl --install

Restart Windows, then open WSL and type:

git clone <your_repo_url>
cd NetworkSafetyChecker
sudo ./install.sh
sudo netscan

---------------------------------------------------------------------
9. Using on Mac
---------------------------------------------------------------------
MacOS cannot run the script directly.  
Mac users can run the tool by:

- Booting from your USB into Linux, or
- Using a Linux virtual machine (VirtualBox, VMware, Parallels)

---------------------------------------------------------------------
10. Explaining the Tool Simply
---------------------------------------------------------------------
This tool checks a computer for weak settings, unsafe network issues, and possible cyber attacks such as ARP spoofing on public Wi-Fi. It gives a score out of 10 and tells the user what should be improved.
